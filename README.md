# financial-calculator

A command-line Python application that performs essential financial and mathematical calculations in one place.

## Features
- Calculate annuity (monthly and continuous compounding)
- Compute monthly mortgage payments
- Estimate retirement investment balance
- Determine time required for an investment to double
- Solve logarithmic equations
- Convert numbers to and from scientific notation

## Requirements
- Python 3.x

## How to Run
1. Open the project folder in VS Code
2. Open the terminal
3. Run:
   python your_filename.py

## Usage
After running the program, choose from the menu:
1. Annuity
2. Mortgage Payment
3. Retirement Balance
4. Time to Double
5. Solve Log Equation
6. Scientific Notation
7. Exit

Enter the required values as prompted.

## Notes
- Enter interest rates in decimal form (e.g., 5% = 0.05)
- Time values are in years unless specified otherwise
